# burger-react
// Installation

Follow the commands below to clone this repository on your system

1. git clone https://github.com/kali151/burger-react.git
2. cd burger-react
3. npm install
4. npm start


// Tech Used

ReactJS 
Redux/Hooks/aux
Firebase


// Features

- Authentication with Email/Password.
- Private Routing
